# 🎯 含胸驼背改善 - 一页版（打印随身带）

---

## 核心口诀（必背）

```
每天3件事：
1️⃣ 收下巴  2️⃣ 开门胸  3️⃣ 收肩胛
```

---

## 动作 1：收下巴 ⭐⭐⭐

```
▢ 坐直/站直
▢ 眼睛平视
▢ 下巴向后收（像挤双下巴）
▢ 保持3秒
▢ 10次 × 3组

⚡ 随时随地可做
```

---

## 动作 2：门框拉伸 ⭐⭐⭐

```
▢ 站门框前
▢ 前臂撑门框，肘与肩同高
▢ 身体前倾
▢ 胸部拉伸感
▢ 60秒 × 2边

💡 没门框？用墙角！
```

---

## 动作 3：收肩胛 ⭐⭐⭐

```
▢ 站直/坐直
▢ 肩膀下沉
▢ 肩胛向中收紧
▢ 想象夹笔
▢ 15次 × 3组

⚡ 坐着也能做
```

---

## 动作 4：YTWL ⭐⭐

```
Y → 双手举成Y，保持3秒
T → 平举成T，保持3秒
W → 弯曲成W，保持3秒
L → 小臂上成L，保持3秒

10-15次 × 3组
```

---

## 动作 5：死虫式 ⭐⭐

```
▢ 仰卧
▢ 双手向天伸直
▢ 腿屈膝抬90度
▢ 右手+左脚同时伸
▢ 换边

10次 × 3组
⚠️ 腰贴地！
```

---

## 动作 6：平板支撑 ⭐

```
▢ 肘撑地
▢ 身体一线
▢ 收紧腹部
▢ 不要塌腰

30-60秒 × 3组
```
