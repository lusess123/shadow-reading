# 含胸驼背 - 精简汇总版

## 核心口诀
每天三件事：收下巴 + 开门胸 + 收肩胛

## 训练等级

### 5分钟版（入门）
```
收下巴 10次×3组 = 3分钟
肩胛收缩 15次   = 1分钟
墙壁天使 30秒   = 1分钟
```

### 15分钟版（推荐）
```
热身 3分钟
  - 猫牛式
  - 肩部环绕

拉伸 5分钟
  - 门框拉伸
  - 颈部拉伸

强化 5分钟
  - YTWL
  - 死虫式

放松 2分钟
  - 婴儿式
```

## 30天计划
```
Week 1: 5分钟/天
Week 2: 10分钟/天
Week 3: 15分钟/天
Week 4: 15分钟/天 + 拍照对比
```

## 效果预期
```
7天  → 自觉挺胸
21天 → 明显改善
60天 → 习惯养成
```

## 核心动作

### 收下巴
- 坐直/站直
- 眼睛平视
- 下巴向后收（像挤出双下巴）
- 保持3秒
- 10次×3组

### 门框拉伸
- 站门框前
- 前臂撑门框，肘与肩同高
- 身体前倾
- 胸部拉伸感
- 60秒×2边

### 收肩胛
- 站直/坐直
- 肩膀下沉
- 肩胛向中收紧
- 想象夹笔
- 15次×3组

## 记住
> 与其追求完美计划，不如现在开始5分钟。
> 60天后，你会感谢现在开始改变的自己！
