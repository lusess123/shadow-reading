## 🎬 动作精解

### 收下巴

```
✅ 坐直/站直
✅ 眼睛平视
✅ 下巴向后收（像挤双下巴）
✅ 保持3秒
✅ 10次×3组
```

### 门框拉伸

```
✅ 门框前站立
✅ 前臂撑门框，肘与肩同高
✅ 身体前倾
✅ 胸部拉伸感
✅ 60秒×2边
```

### 收肩胛

```
✅ 站直/坐直
✅ 肩膀下沉
✅ 肩胛向中收紧
✅ 想象夹笔
✅ 15次×3组
```

### 死虫式

```
✅ 仰卧
✅ 双手向天伸直
✅ 腿屈膝抬90度
✅ 伸右手+左脚
✅ 换边
✅ 10次×3组
```

### 平板支撑

```
✅ 肘撑地
✅ 身体一线
✅ 收紧腹部
✅ 不要塌腰
✅ 30-60秒×3组
```
