# 4️⃣ 动作详解

## ⭐ 收下巴

```
✅ 坐直/站直
✅ 眼睛平视
✅ 下巴向后收（像挤双下巴）
✅ 保持3秒
✅ 10次×3组

⚡ 随时随地可做
```

## ⭐ 门框拉伸

```
✅ 站门框前
✅ 前臂撑门框，肘与肩同高
✅ 身体前倾
✅ 胸部拉伸感
✅ 60秒×2边

💡 没门框？用墙角！
```

## ⭐ 收肩胛

```
✅ 站直/坐直
✅ 肩膀下沉
✅ 肩胛向中收紧
✅ 想象夹笔
✅ 15次×3组
```

## YTWL

```
Y: 双手举成Y，保持3秒
T: 平举成T，保持3秒
W: 弯曲成W，保持3秒
L: 小臂上成L，保持3秒

10-15次×3组
```

## 死虫式

```
✅ 仰卧
✅ 双手向天伸直
✅ 腿屈膝抬90度
✅ 右手+左脚同时伸
✅ 换边

10次×3组
⚠️ 腰贴地！
```

## 平板支撑

```
✅ 肘撑地
✅ 身体一线
✅ 收紧腹部
✅ 不要塌腰

30-60秒×3组
```

# 7️⃣ 避坑指南

```
❌ 拉伸越疼越好    → ✅ 轻微牵拉感即可
❌ 只拉伸不强化    → ✅ 两者缺一不可
❌ 动作做得很快    → ✅ 慢动作感受肌肉
❌ 一周就放弃     → ✅ 6-8周才见效果
❌ 用力时憋气     → ✅ 用力呼气收缩吸气
❌ 塌腰做平板支撑 → ✅ 收紧腹部
❌ 三天打鱼两天晒网 → ✅ 每天5分钟>每周2小时
```
