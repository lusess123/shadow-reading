# 含胸驼背改善 - 一页纸终极版

---

## 每天三件事

```
1️⃣ 收下巴   10次×3组
2️⃣ 开门胸   60秒×2边
3️⃣ 收肩胛   15次×3组
```

---

## 动作图解

```
收下巴：下巴向后收，像挤双下巴

门框拉伸：前臂撑门框，身体前倾

收肩胛：肩胛向中收，像夹笔

YTWL：Y T W L 保持3秒

死虫式：腰贴地，伸手脚

平板支撑：收紧腹，不塌腰
```

---

## 5分钟版

```
收下巴 10次×3组 = 3分钟
肩胛收缩 15次   = 1分钟
墙壁天使 30秒   = 1分钟
共5分钟
```

---

## 15分钟版（推荐）

```
热身 3分钟
  猫牛式+肩部环绕

拉伸 5分钟
  门框拉伸+中束拉伸

强化 5分钟
  YTWL+死虫式

放松 2分钟
  婴儿式
```

## 30天

Week1: 5分钟×5天

```

## 避坑

```
❌ 疼=有效   → ✅ 轻微牵拉
❌ 只拉伸    → ✅ 拉伸+强化
❌ 追求快   → ✅ 慢动作
❌ 一周放弃 → ✅ 6-8周
```

```
