## 🎯 核心动作详解

### 1️⃣ 收下巴

```
步骤：
1. 坐直或站直
2. 眼睛平视前方
3. 下巴轻轻向后收
4. 感觉像挤出双下巴
5. 保持3秒
6. 重复10次 × 3组

要点：
- 不要低头！
- 感受颈部后侧发力
- 每天任何时候都可以做
```

### 2️⃣ 开门胸（门框拉伸）

```
步骤：
1. 站在门框前
2. 前臂撑住门框，肘部与肩同高
3. 身体慢慢向前倾
4. 胸部有拉伸感
5. 保持60秒，换边

替代方案：
- 没有门框 → 用墙角
- 办公室 → 椅子背拉伸
- 床上 → 抱枕拉伸
```

### 3️⃣ 收肩胛

```
步骤：
1. 站直或坐直
2. 肩膀向后向下沉
3. 肩胛骨向中间收紧
4. 想象要夹住一支笔
5. 保持3秒
6. 重复15次 × 3组

进阶：
- 拿弹力带阻力
- 负重练习
```

## 三个难度等级

### 🌱 等级1：微运动（5分钟）

适合：超级忙、刚开始运动的人

| 动作 | 次数 | 时间 |
|------|------|------|
| 收下巴 | 10次×3组 | 3分钟 |
| 肩胛收缩 | 15次 | 1分钟 |
| 墙壁天使 | 30秒 | 1分钟 |

### 🌿 等级2：标准版（15-20分钟）推荐

适合：大多数人

```
流程：热身(3min) → 拉伸(7min) → 强化(8min) → 放松(2min)

🔥 热身
  - 猫牛式：1分钟
  - 肩部环绕：1分钟

🔸 拉伸
  - 门框胸部拉伸：2×60秒
  - 胸部中束拉伸：2×60秒
  - 颈部拉伸：2×30秒

💪 强化
  - YTWL训练：3组
  - 弹力带划船：3组
  - 死虫式：3组
  - 平板支撑：3组×30秒

🧘 放松
  - 婴儿式：1分钟
  - 仰卧放松：1分钟
```

### 🌳 等级3：加强版（30+分钟）

适合：追求效果、规律运动者

在标准版基础上增加：
- 哑铃划船
- 游泳（每周2-3次）
- 普拉提课程

✅ 通勤路上 → 靠墙站

✅ 只做拉伸
✅ 站着收下巴

【正向反馈法】

## 🥗 饮食建议（辅助改善）

虽然体态主要靠训练，但饮食也很重要！

### 推荐食物

| 食物 | 功效 |
|------|------|
| 蛋白质（鸡胸肉、鱼、豆制品） | 修复肌肉 |
| 维生素D（晒太阳、鱼油） | 骨骼健康 |
| 钙（牛奶、豆腐、芝麻） | 骨骼强健 |
| 镁（坚果、深绿叶菜） | 肌肉放松 |
| Omega-3（深海鱼、亚麻籽） | 减少炎症 |

### 少吃/避免

| 食物 | 原因 |
|------|------|
| 加工食品 | 可能加剧炎症 |
| 高糖食品 | 影响肌肉恢复 |
| 咖啡因过量 | 可能影响睡眠 |
| 酒精 | 影响肌肉修复 |

### 习惯建议

- 多喝水（每天2L）
- 均衡饮食
- 训练后补充蛋白质
- 保证7-8小时睡眠

## 常见错误（避坑指南）

```
❌ 拉伸越疼越好
✅ 轻微牵拉感即可

❌ 只拉伸不强化
✅ 两者缺一不可

❌ 动作做得很快
✅ 慢动作，感受肌肉

❌ 坚持一周就放弃
✅ 6-8周才见效果

❌ 用力时憋气
✅ 用力呼气，收缩吸气

❌ 塌腰做平板支撑
✅ 收紧腹部

❌ 三天打鱼两天晒网
✅ 每天5分钟 > 每周2小时
```
