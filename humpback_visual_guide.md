## 🎯 动作图解

### 收下巴

```
正面：
   ○
   │
   ▽←下巴回收

侧面：
○→▽  错误（头前倾）

  ○    正确
  ↓
  ▽←下巴回收
```

### 门框拉伸

```
    ┌────┐
    │门框│
 ←──│    │───→ 身体前倾方向

    手←──────→前臂
    ↓
  身体倾斜
```

### 收肩胛

```
错误：              正确：
  ▽  ←肩膀耸起      ▽←肩膀下沉
      ▓▓             ▓▓
     ▓▓▓▓           ▓▓▓▓←肩胛收紧
    ▓▓▓▓▓▓         ▓▓▓▓▓▓←夹紧
```

### YTWL

```
Y:    ↑
     /▕
    /▕ \

T:   ———
    /▕ \

W:   \/
    /▕ \
    ▓▓▓

L:   ▕
    ▕▘
```

### 死虫式

```
仰视：
    手←——————→脚
      ↖      ↗
       身体
       屈膝90°

侧视：
    ▽  ←头
    █  ←身体
    █  ←腿
```

### 平板支撑

```
错误：           正确：
  █            █
   █ █          ███
    █           █
   塌腰        直线
```

## 📐 正确角度

### 收下巴

```
眼睛平视前方
下巴垂直向后收
头部位置：耳垂在肩膀正上方
```

### 门框拉伸

```
前臂与地面平行
肘部与肩同高
身体前倾30-45度
```

### 收肩胛

```
肩膀下沉
肩胛向中靠拢
想象夹一支笔
```

### 死虫式

```
腰部贴地
腿抬90度
手向上伸直
```

### 平板支撑

```
肘在肩正下方
身体呈直线
臀部不翘起
```

## 🔴 错误示范

### 收下巴

```
❌ 低头
❌ 头向前伸
❌ 下巴向下收
```

### 门框拉伸

```
❌ 手臂太高
❌ 没有前倾
❌ 耸肩
```

### 收肩胛

```
❌ 肩膀耸起
❌ 只动手臂
❌ 动作太快
```

### 死虫式

```
❌ 腰部抬起
❌ 手脚乱晃
❌ 憋气
```

### 平板支撑

```
❌ 塌腰
❌ 翘臀
❌ 头仰着
```

## 🏷️ 训练顺序

```
1. 热身（3分钟）
   - 猫牛式
   - 肩部环绕

2. 拉伸（5-7分钟）
   - 门框拉伸
   - 胸部拉伸

3. 强化（5-8分钟）
   - YTWL
   - 死虫式

4. 放松（2分钟）
   - 婴儿式
```

### 收下巴

- 眼睛始终平视
- 下巴垂直向后收

- 保持呼吸

### 收肩胛

- 肩膀主动下沉
- 感受肩胛收紧
- 想象夹笔
