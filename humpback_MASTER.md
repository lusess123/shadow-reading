# 1️⃣ 什么是含胸驼背

## 定义

- **含胸（圆肩）**：肩膀向前内收
- **驼背（胸椎后凸）**：胸椎向后突出
- **头前倾**：头部向前移

## 原因

- 久坐办公
- 不良姿势
- 缺乏运动
- 肌肉失衡

# 2️⃣ 如何自测

## 简单测试

```
[ ] 侧面拍照，耳垂在肩膀前方
[ ] 贴墙站，后脑勺不能贴墙
[ ] 手臂不能从身后向上抬起

0-1个：很好
2-3个：轻度
4-5个：中度
6个：重度
```

# 3️⃣ 核心动作

## ⭐ 收下巴

```
步骤：
1. 坐直/站直
2. 眼睛平视
3. 下巴向后收（像挤出双下巴）
4. 保持3秒
5. 重复10次×3组

要点：不要低头，感受颈部后侧发力
```

## ⭐ 门框拉伸

```
步骤：
1. 站门框前
2. 前臂撑门框，肘与肩同高
3. 身体前倾
4. 胸部拉伸感
5. 保持60秒×2边

替代：墙角、椅子背
```

## ⭐ 收肩胛

```
步骤：
1. 站直/坐直
2. 肩膀下沉
3. 肩胛向中收紧
4. 想象夹笔
5. 15次×3组
```

## YTWL

```
Y: 双手举成Y，保持3秒
T: 平举成T，保持3秒
W: 弯曲成W，保持3秒
L: 小臂上成L，保持3秒

10-15次×3组
```

## 死虫式

```
步骤：
1. 仰卧
2. 双手向天伸直
3. 腿屈膝抬90度
4. 右手+左脚同时伸
5. 换边

10次×3组
重点：腰贴地！
```

## 平板支撑

```
步骤：
1. 肘撑地
2. 身体一线
3. 收紧腹部
4. 不要塌腰

30-60秒×3组
```
