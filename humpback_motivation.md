## 🧠 心态建设

### 为什么会放弃？

```
原因1：目标太高
  ❌ 每天30分钟
  ✅ 每天5分钟

原因2：看不到效果
  ❌ 一周没变化就放弃
  ✅ 记录对比，耐心等待

原因3：太累了
  ❌ 不练了
  ✅ 5分钟也能做

原因4：太难了
  ❌ 动作太复杂
  ✅ 从最简单的开始
```

## 🎯 心理技巧

### 1. 绑定习惯

```
把训练绑定到已有的习惯上：

起床 → 收下巴
洗澡 → 拉伸
追剧 → 门框拉伸
睡前 → 平板支撑
```

### 2. 降低门槛

```
状态不好时：
✅ 至少1分钟
✅ 只做拉伸
✅ 躺着也能练
❌ 不要"不做了"
```

### 3. 正向反馈

```
记录进步：
✅ 每天记录
✅ 每周拍照
✅ 每月对比

给自己奖励：
✅ 7天奖励
✅ 21天奖励
✅ 30天大奖励
```

### 4. 视觉化

```
想象：
✅ 变帅/变美的样子
✅ 穿衣服更好看
✅ 更有气质
✅ 远离肩颈酸痛
```

## 😔 低谷期怎么办？

### 不想动时

```
问自己：
"真的连5分钟都没有吗？"
"真的累到动不了吗？"

答案是：
"我可以！"
"动一下就舒服了！"
```

### 没效果时

```
问自己：
"真的没效果吗？"
"我有认真做吗？"

检查：
✅ 动作标准吗？
✅ 每天坚持了吗？
✅ 姿势有改善吗？

答案是：
"坚持就是胜利！"
```

### 想放弃时

```
问自己：
"为什么开始？"
"想要什么样的结果？"

提醒自己：
"60天后，我会感谢现在坚持的自己！"
```
